## ✨ 기능
- [X] 거상 자동 로그인 및 게임 실행<br>
- [X] 다계정 & 다클라(최대 3개) 지원<br>
- [X] 빠르고 용량 적은 다클라 생성<br>
- [X] 게임 설치 및 패치 지원<br>
- [X] 본섭, 테섭, 천라섭 지원<br>
- [X] 게임 공지사항, 이벤트 표시(종료 임박 시 강조)<br>
- [X] 브라우저 페이지에서 간편하게 이벤트 보상 수령<br>
- [ ] (개발 중) 향상된 마우스 가두기 (Contributed By [@chlrhkd4](https://github.com/chlrhkd4))
- [ ] (개발 중) 향상된 창 전환 (1,2,3클 전환을 빠르고 쉽게)
- [ ] (개발 중) 플래너 (휴면계정 및 생산시설 일정 관리, 공성/국가토벌전/미궁 알림 등)
