거상스테이션 개인정보처리방침

문서 버전: 1.0.0
시행일: 2026-03-10
최종 수정일: 2026-03-10

거상스테이션(이하 "앱")은 이용자의 개인정보를 중요하게 생각하며, 앱이 처리하는 개인정보와 그 처리 방식에 대해 다음과 같이 안내합니다.
이 방침은 앱 자체가 처리하는 개인정보를 기준으로 작성되며, 거상 공식 홈페이지/게임 서비스, GitHub, Microsoft Store 등 외부 서비스의 개인정보 처리에는 각 서비스의 정책이 별도로 적용됩니다.

1. 처리하는 개인정보 항목

앱은 다음 정보를 처리할 수 있습니다.

[이용자가 직접 입력하거나 저장하는 정보]
- 거상 계정 아이디
- 계정 별명, 그룹명
- 비밀번호
- 서버 선택 정보
- 게임 설치 경로, 다클라 사용 여부, 패치 관련 옵션, 프리셋 정보 등 실행/설정 정보
- 브라우저 즐겨찾기 이름, URL, 파비콘 캐시 정보
- 최초 실행 여부, 선택한 프리셋, 개발자 도구 사용 여부 등 앱 상태 정보

[서비스 이용 과정에서 일시적으로 처리되는 정보]
- 거상 공식 홈페이지 로그인 상태 확인을 위한 세션 정보(memberID 쿠키 값 등)
- OTP 인증번호
- 게임 실행 절차에서 거상 공식 홈페이지가 로컬 루프백(127.0.0.1)으로 전달하는 실행용 계정 정보(id, pw, accountId 등)

[외부 서비스 접속 과정에서 발생하는 정보]
- 거상 공식 홈페이지 이벤트/공지 조회를 위한 웹 요청 정보
- GersangStation GitHub Discussions 공지 조회를 위한 웹 요청 정보
- Microsoft Store 업데이트 확인 및 설치 요청 시 필요한 스토어 요청 정보

앱은 주민등록번호, 휴대전화번호, 이메일 주소, 결제정보를 앱 자체 저장소에 별도로 수집하거나 저장하지 않습니다.

2. 개인정보 처리 목적

앱은 다음 목적 범위에서 개인정보를 처리합니다.

- 저장한 계정의 선택, 관리 및 로그인 자동화
- OTP 입력 보조 및 게임 실행 절차 진행
- 설치 경로, 패치 설정, 다클라 설정, 프리셋 등 런처 기능 제공
- 브라우저 즐겨찾기 및 앱 내부 WebView 기능 제공
- 거상 공식 이벤트/공지, GersangStation 공지, Microsoft Store 업데이트 상태 확인
- 사용자가 선택한 서버와 최근 사용 상태 복원

3. 개인정보의 저장 위치 및 보관 기간

[로컬 앱 데이터]
- 계정 목록, 설정값, 프리셋, 즐겨찾기, 파비콘 캐시 등은 Windows 앱 전용 로컬 저장소(LocalFolder)에 저장됩니다.
- 해당 정보는 이용자가 직접 삭제하거나 앱 데이터를 초기화할 때까지 보관될 수 있습니다.

[비밀번호]
- 비밀번호는 일반 텍스트 파일에 저장하지 않고 Windows 자격 증명 관리자(PasswordVault/Credential Locker)에 저장합니다.
- 비밀번호는 계정을 앱에서 삭제하면 함께 제거되도록 설계되어 있습니다.
- 그 외 경우에는 이용자가 직접 계정을 삭제하거나 운영체제 자격 증명 저장소를 정리하기 전까지 남아 있을 수 있습니다.

[OTP 인증번호 및 실행용 계정 정보]
- OTP 인증번호는 인증 절차 중 일시적으로만 사용되며 별도 파일로 저장하지 않습니다.
- 게임 실행용 계정 정보(id, pw, accountId 등)는 게임 시작 절차 수행을 위해 일시적으로 처리되며, 별도 영구 저장소에 기록하지 않습니다.

[거상 공식 홈페이지 세션 정보]
- 거상 공식 홈페이지와의 로그인 상태 확인을 위해 WebView 세션 정보를 참조할 수 있습니다.
- 세션 쿠키의 생성·보관·만료는 거상 공식 홈페이지 및 WebView2 동작에 따라 달라질 수 있습니다.

4. 제3자 제공

앱은 개발자의 별도 서버나 광고 사업자에게 이용자의 개인정보를 판매하거나 제공하지 않습니다.

다만, 이용자가 앱 기능을 사용하는 과정에서 다음 외부 서비스와 직접 통신할 수 있습니다.

- 거상 공식 홈페이지/게임 서비스: 로그인, OTP 인증, 로그아웃, 게임 시작
- GitHub: GersangStation 공지사항 조회
- Microsoft Store: 앱 업데이트 확인 및 설치

이 경우 각 서비스 제공자가 처리하는 정보와 그 방식은 해당 서비스의 개인정보처리방침 및 이용약관에 따릅니다.

5. 처리위탁

앱은 개발자 자체 서버 운영을 위한 별도 개인정보 처리위탁을 확인 가능한 범위 내에서 두고 있지 않습니다.
다만 Windows, Microsoft Store, WebView2, GitHub, 거상 공식 서비스 등 외부 플랫폼은 각자 독립된 서비스 제공자로서 자체 정책에 따라 정보를 처리할 수 있습니다.

6. 개인정보의 파기

앱은 개인정보 처리 목적이 달성되었거나 이용자가 삭제한 정보에 대해 가능한 범위에서 지체 없이 파기합니다.

- 계정 삭제 시: 저장된 계정 목록과 연결된 비밀번호를 함께 정리합니다.
- 즐겨찾기 삭제 시: 관련 즐겨찾기 정보와 파비콘 캐시를 정리할 수 있습니다.
- 설정 변경 또는 초기화 시: 기존 설정 파일은 새 값으로 대체되거나 삭제될 수 있습니다.
- OTP 인증번호 등 일시 정보: 절차 종료 후 메모리에서 해제되는 것을 원칙으로 합니다.

7. 이용자의 권리와 행사 방법

이용자는 앱 내 기능을 통해 다음과 같은 권리를 행사할 수 있습니다.

- 계정 추가, 수정, 삭제
- 프리셋 및 즐겨찾기 추가, 수정, 삭제
- 서버 선택 및 각종 런처 설정 변경

거상 공식 홈페이지 계정 자체의 개인정보, 결제정보, 회원정보 정정·삭제 등은 거상 공식 서비스 운영자에게 직접 요청해야 합니다.

8. 자동 수집 장치의 설치·운영 및 거부

앱은 자체 광고 SDK나 자체 분석용 쿠키를 사용하지 않습니다.
다만 앱 내부 WebView에서 거상 공식 홈페이지를 표시하는 과정에서 해당 서비스가 쿠키 또는 유사 저장소를 사용할 수 있으며, 앱은 로그인 상태 확인과 게임 실행 흐름 유지를 위해 memberID 쿠키 값을 확인할 수 있습니다.

9. 개인정보의 안전성 확보 조치

앱은 다음과 같은 조치를 적용합니다.

- 비밀번호는 Windows 자격 증명 관리자에 저장
- 일반 설정 정보는 앱 전용 로컬 저장소에 저장
- 게임 시작에 필요한 로컬 통신은 127.0.0.1 루프백 주소로 한정
- 개발자 자체 원격 서버에 계정/비밀번호를 별도 저장하지 않음

10. 외부 서비스 안내

앱은 다음 외부 서비스와 연동될 수 있습니다.

- 거상 공식 홈페이지: https://www.gersang.co.kr
- GersangStation GitHub: https://github.com/byungmeo/GersangStation
- Microsoft Store 및 Windows 플랫폼 서비스

각 외부 서비스는 자체 개인정보처리방침을 가질 수 있으며, 외부 서비스 이용에 관한 사항은 해당 서비스 정책을 함께 확인하시기 바랍니다.
